Opdracht 3

Waarom staat de tekst in de blokken horizontaal gecentreerd?

Dit komt door div { text-align:center; } in de css dat ervoor zorgt dat de tekst gecentreerd staat.

Waarom heeft vertical-align:center geen effect?

De vertical-align:center heeft geen effect omdat we de stuctuur van onze html al hebben opgesteld in de .area met with en height om 100px.

Wat doet vertical-align?

Centreert de elementen i.p.v. de tekst.

Waarom krijgt de regel voor class 'tall' voorrang?

Omdat specifity van tall en area gelijk is maar omdat tall achter area komt wordt tall als laatst uitgevoerd.
