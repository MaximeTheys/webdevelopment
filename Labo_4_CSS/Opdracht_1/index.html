1. Als je je display van .area op block zet komt alles ander elkaar te staan.
  2. Als je je display van .area op inline zet komt alles naast elkaar te staan.
  3. Als je je display van .area op inline-block zet komt alles per rij onder elkaar te staan, de grootte van de rij hangt af van hoe groot de pagina is.

Voor welke waarden worden de opgegeven width en height genegeerd?

voor inline

Voor welke waarden worden de elementen als woorden op een pagina gezet?

voor block, inline-block
